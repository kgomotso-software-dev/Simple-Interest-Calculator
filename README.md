# Simple-Interest-Calculator
A simple command-line program to calculate simple interest.

## Formula
Simple Interest = (P × R × T) / 100

Where:
- **P** = Principal amount (initial investment or loan amount)
- **R** = Annual interest rate (in percentage)
- **T** = Time period (in years)

## How It Works

1. User enters the principal amount
2. User enters the annual interest rate
3. User enters the time period in years
4. Program calculates and displays the simple interest

## Example

**Input:**
Principal = 1000
Rate = 5%
Time = 3 years

**Calculation:**
Simple Interest = (1000 × 5 × 3) / 100 = 150
